# Homebrew Tap for Powertools

Agentic workflow toolkit with persistent memory and task tracking for AI-assisted development.

## Install

```bash
brew tap fincognition/powertools
brew install powertools-ai
```

## Requirements

- Apple Silicon Mac (M1/M2/M3/M4)
- macOS 14+ (Sonoma)

## Setup

After installation:

```bash
# Initialize (one-time) - installs embedding daemon, downloads model
pt init

# Initialize a project
cd your-project
pt project-init -n myproject
```
